Healthcare Patient Waitlist Dashboard
Overview
The Healthcare Patient Waitlist Dashboard is an interactive data visualization project designed to analyze patient waiting list trends across different medical specialities and case types. The dashboard provides clear insights into patient wait times, case distribution, and monthly trends, enabling healthcare administrators to make data-driven decisions.

* Key Features
Waitlist Monitoring: Displays the median and average patient waiting times across different specialities.
Case Type Analysis: Breaks down patient data into Outpatient, Day Case, and Inpatient categories.
Age Group Insights: Visualizes patient wait times across different age profiles (0–15, 16–64, 65+).
Time Band Analysis: Shows patient distribution based on waiting periods such as 0–3 months, 3–6 moths, 6–9 months, and beyond.
Monthly Trend Analysis: Tracks historical trends in patient waitlists over time.
Interactive Filters: Users can filter the dashboard by archive date, case type, speciality, and statistical measure (average or median).

* Key Insights Provided
 Comparison between current and previous year waitlist totals.
 Identification of specialities with higher patient wait times.
 Analysis of long-term patient waiting patterns.
 Performance tracking for healthcare services.
